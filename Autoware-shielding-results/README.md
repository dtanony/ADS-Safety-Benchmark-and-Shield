## Shielded Autoware Experiment Results

This folder contains the trace files recorded during the experiments with the shielded Autoware (Autoware integrated with a controller safety shield).
The structure of the experiments is the same as that of the baseline Autoware experiments (see [Autoware-baseline-results](../Autoware-baseline-results/)).

To replicate the shielded Autoware experiments, please follow the instructions in the [Autoware-baseline-results](../Autoware-baseline-results#experiment-reproduction) folder.
